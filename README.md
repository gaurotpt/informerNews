# Informer - Real-Time News Portal 📰

Informer is a modern, responsive, and real-time news web application. It fetches live news headlines and trending YouTube Shorts to keep users updated with the latest happenings around the world.

## 🚀 Features
- **Live News:** Fetches real-time headlines using NewsAPI.
- **Trending Shorts:** Integrates YouTube Data API to display latest trending news shorts.
- **Dark Mode:** User-friendly toggle to switch between light and dark themes.
- **Search Functionality:** Users can search for news by topic, city, or category.
- **Responsive Design:** Mobile-first approach for seamless viewing on all devices.

## 🛠 Tech Stack
- **HTML5:** For structured layout.
- **CSS3:** For modern and responsive styling.
- **JavaScript (ES6):** For API handling and DOM manipulation.
- APIs: - [NewsAPI](https://newsapi.org/) for news content.
    - [YouTube Data API v3](https://developers.google.com/youtube/v3) for video content.

How it works
